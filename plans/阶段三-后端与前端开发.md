**流式输出实现**（关键代码）：

```python
import httpx

def stream_query(question: str):
    """流式调用 RAG API"""
    with httpx.stream(
        "POST",
        "http://localhost:8000/api/v1/query",
        json={"question": question, "stream": True},
        timeout=30.0,
    ) as response:
        for line in response.iter_lines():
            if line.startswith("data: "):
                data = json.loads(line[6:])
                if data.get("type") == "token":
                    yield data["content"]
                elif data.get("type") == "done":
                    # 附加来源信息
                    yield ("__sources__", data["sources"])
```

## 阶段三常见阻塞点

| 阻塞 | 解决方式 |
|------|----------|
| FastAPI 路由注册后 404 | 检查 `prefix` 是否拼错；检查 `include_router` 是否遗漏 |
| Streamlit 连接后端失败 | 确认后端已启动；检查端口号；检查 CORS 配置 |
| 文件上传后处理超时 | 大文件解析 + 向量化可能需几十秒，前端加 loading 状态 |
| 流式输出乱码 | SSE 的 `data:` 前缀格式要正确；检查 JSON 序列化 |
| st.session_state 跨页面不共享 | Streamlit 多页面默认共享 session_state，确认 key 名一致 |
