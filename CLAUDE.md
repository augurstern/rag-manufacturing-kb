# CLAUDE.md

本文件为 Claude Code（claude.ai/code）在此仓库中工作时提供指导。

## 仓库概述

这是一个**处于早期规划阶段的 RAG（检索增强生成）项目**。仓库将从一份需求文档逐步演进为一个可落地的企业级内部知识库系统，面向制造业集团场景。当前尚无任何可执行代码——仅有初始规划/参考文档。

## 核心文档

[docs/企业级RAG完整设计方案.md](docs/企业级RAG完整设计方案.md)

这是**项目蓝图**，描述了需要构建的完整系统愿景。本项目实际实现其简化版（FastAPI + ChromaDB + DeepSeek API），面试中可用完整文档展示系统设计能力：

- **技术栈**：Dify（私有化 LLMOps）+ Milvus（分布式向量库）+ PGVector + BGE-M3（私有化 Embedding）+ 通义千问 72B（本地部署大模型）+ RAGFlow（OCR/文档解析），Docker Compose 起步，后期迁移 K8s
- **6 个实施阶段**：需求调研 → POC 验证 → 数据 ETL 流水线 → Dify 定制开发 → 测试与合规验收 → 上线运维
- **企业硬性约束**：全内网离线环境、RBAC 多租户隔离、扫描件 OCR 识别、审计日志、单次问答≤3s、200 人并发

## 项目当前状态

**阶段：阶段一 — 环境搭建与技术储备（06.30 - 07.12）。** 项目骨架已建立，正在学习 Docker/FastAPI/Streamlit 基础。后续 5 阶段执行计划见 `.claude/plans/` 目录。
